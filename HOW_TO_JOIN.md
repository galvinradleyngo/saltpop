# How to Join the Saltpop Space

Welcome! Saltpop is a virtual collaboration space where you can chat, share screens, and work together in real time.

## Joining

1. Open the invite link in any modern browser (Chrome, Edge, Firefox, Safari).  
   No app download is required.
2. Enter your name and choose an avatar.
3. Allow microphone and camera access when prompted.  
   You can mute or turn off your camera at any time once you're inside.
4. Click **Join** – you're in!

## Supported devices & browsers

| Platform | Browser |
|---|---|
| Windows / macOS / Linux | Chrome 90+, Edge 90+, Firefox 90+ |
| iOS (iPhone / iPad) | Safari 15+ |
| Android | Chrome 90+ |

> **Tip:** For the best experience, use a device with a camera and microphone.
> Headphones help reduce echo.

## Controls inside the space

| Action | How |
|---|---|
| Mute / unmute | Click the microphone icon in the toolbar |
| Camera on / off | Click the camera icon in the toolbar |
| Move around | Click and drag on the canvas |
| Share a link / file | Use the **+** button in the toolbar |
| Leave | Close the browser tab |

## Troubleshooting

**I can't hear anyone.**  
→ Check that your browser has microphone permission (look for a camera/mic icon in the address bar).

**My video isn't showing.**  
→ Make sure no other app is using your camera, then refresh the page.

**The link doesn't work.**  
→ Try opening the link in an incognito/private window, or clear your browser cache and retry.

**On iOS: no audio/video.**  
→ Only Safari is fully supported on iOS. Please use Safari instead of Chrome or Firefox on iPhone/iPad.
